# Todo List 應用程式

一個簡潔的待辦事項管理應用程式，使用純 JavaScript 開發，並整合 Google Sheets 作為雲端資料庫。

## ✨ 功能特色

- 📝 新增/刪除待辦事項
- ✅ 勾選完成狀態
- 📊 依優先權排序（無 > 高 > 中 > 低）
- 🎨 優先權色彩標示（紅/橘/綠/灰）
- 📅 自動記錄建立日期
- ☁️ Google Sheets 雲端同步
- ⚙️ 可自訂 Google Script URL

## 🚀 快速開始

### 1. 建立 Google Sheets 試算表

1. 開啟 [Google Sheets](https://sheets.google.com) 並建立新的試算表
2. 在第一列（標題列）依序輸入以下欄位名稱：

| 欄位 | 名稱 | 說明 |
|------|------|------|
| A1 | `紀錄日期` | 待辦事項建立時間 |
| B1 | `待辦事項` | 待辦事項名稱 |
| C1 | `優先權` | 可填入：高、中、低 |
| D1 | `完成日期` | 完成時間（選填） |
| E1 | `檢查狀態` | 勾選框（會自動建立） |
| F1 | `備註` | 待辦事項描述 |
| G1 | `ID` | 系統自動產生的識別碼 |

> 💡 **提示**：只需要輸入標題列即可，資料會由應用程式自動填入。

### 2. 設定 Google Apps Script

1. 在試算表中，點選「**擴充功能**」>「**Apps Script**」
2. 刪除預設程式碼，複製 `google-apps-script.js` 的內容貼入
3. 點選「**儲存**」（Ctrl+S）
4. 點選「**部署**」>「**新增部署**」
5. 選擇「**網頁應用程式**」，設定：
   - 執行身分：**我**
   - 誰可以存取：**所有人**
6. 點選「**部署**」，完成授權流程
7. 複製顯示的 **Web App URL**

### 3. 設定前端應用

1. 開啟 `index.html`
2. 點擊右上角 ⚙️ 設定按鈕
3. 貼上 Web App URL 並儲存

### 4. 💻 本機桌面端使用方式

本應用程式設計為純靜態網頁，無需安裝 Node.js 或架設伺服器即可在桌面端使用：

1. **直接開啟**：找到資料夾中的 `index.html` 檔案，雙擊直接以瀏覽器（如 Chrome, Edge）開啟。
2. **建立捷徑**：右鍵點擊 `index.html` >「傳送到」>「桌面（建立捷徑）」，之後即可像應用程式一樣快速啟動。
3. **資料同步**：只要設定好 Google Script URL，無論在哪台電腦開啟此檔案，都能同步存取您的雲端待辦清單。

## 📊 Google Sheets 欄位對應

| 欄位 | 內容 |
|------|------|
| A - 紀錄日期 | 自動填入 `YYYY-MM-DD HH:mm:ss` |
| B - 待辦事項 | Todo 名稱 |
| C - 優先權 | 高/中/低 |
| D - 完成日期 | （暫未使用） |
| E - 檢查狀態 | 勾選狀態 |
| F - 備註 | Todo 描述 |
| G - ID | 唯一識別碼（自動產生） |

## ⌨️ 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Enter` | 在名稱輸入框送出 |
| `Ctrl+Enter` | 在任意輸入框送出 |

## 🔒 安全性說明

目前 Web App 設定為「所有人都能存取」，URL 本身即為存取憑證。請勿將 URL 分享給不信任的人。
